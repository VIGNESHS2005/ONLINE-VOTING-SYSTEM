The Online Voting System using Blockchain is a secure and transparent digital platform designed to conduct college elections through blockchain technology. This system allows students to vote online while ensuring that the voting process is tamper-proof, transparent, and highly secure.
In this system, each student logs in using their unique college credentials such as student ID or registration number. Once authenticated, the student can view the list of candidates and cast their vote. The vote is then stored as a block in the blockchain network, making it immutable and impossible to modify or delete.
Blockchain technology ensures that every vote is securely recorded and verified across the network, preventing fraud, duplicate voting, or manipulation of results. Each student is allowed to vote only once, and the system maintains anonymity while ensuring the integrity of the election.
The system also includes an admin module where administrators can manage candidate registration, start or end the voting process, and monitor the election status. Once voting is completed, the system automatically counts the votes stored in the blockchain and displays the results instantly.
By using blockchain technology, the Online Voting System provides higher security, transparency, reliability, and trust compared to traditional voting methods. It also reduces manual work, minimizes errors, and allows students to participate in elections easily from anywhere.
